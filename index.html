Hoe werkt het?

Klik Start sessie Vanuit deze site of Obsidian project note
n8n checkt bestaande sessie Als er al een claude.ai URL in de temp file staat → direct redirect
tmux sessie aanmaken of hergebruiken 3 windows: claude / shell / git — in de project folder
claude remote-control starten Output → /tmp/claude-rc-{sessie}.txt op de server
URL pollen (max 60s) Elke 5 seconden checken of de claude.ai URL er al in staat
Browser redirect → claude.ai Direct in je project, zonder SSH

Workflows

Start Project Session actief
/webhook/start-project?session=X&folder=Y&mode=normal|dangerous
Herstart of start een sessie. mode=dangerous voegt --dangerously-skip-permissions toe.
Stop Project Session actief
/webhook/stop-project?session=X
Kill tmux sessie en verwijdert de temp file.
New Project Session actief
/webhook/new-project?name=X
Maakt GitHub repo aan (private), clonet naar de server, start sessie.

Server

Host: root@192.168.1.34 Projects root: /root/projects/{naam} Temp files: /tmp/claude-rc-{sessie}.txt tmux windows: claude / shell / git

Project toevoegen aan deze site

Open index.html en voeg een object toe aan de projects array:

{ name: 'project-naam', // ook de tmux sessie naam label: 'Leesbare naam', folder: '/root/projects/project-naam', github: 'https://github.com/lutjebroeker/project-naam', // optioneel vercel: 'https://project.vercel.app', // optioneel }
